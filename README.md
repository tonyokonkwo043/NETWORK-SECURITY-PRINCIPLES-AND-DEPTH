# Network Security Principles & Defense

## 📌 Overview
This project demonstrates core network security principles and practical simulations of defensive measures, including firewall rules, access controls, and intrusion detection concepts.

---

## 🛠️ Security Principles Covered
- **Defense in Depth:** Layered security with multiple controls
- **Least Privilege:** Limiting user/network access to minimum required
- **Network Segmentation:** Using subnets and VLANs to isolate traffic
- **Firewalls & ACLs:** Simulating packet filtering and traffic rules
- **Intrusion Detection:** Monitoring network activity for suspicious behavior
- **Logging & Monitoring:** Tracking events for security auditing

---

## 🛠️ Tools / Technologies
- Python 3.x
- iptables / firewall simulation
- Network simulation via Python scripts
- Diagrams for conceptual visualization

---

## 📂 Project Files
- `firewall_simulation.py` → Demonstrates basic firewall rule simulation
- `access_control.py` → Simulates user and network access policies
- `intrusion_detection_demo.py` → Simple IDS logic using Python
- `diagrams/network_security_diagram.png` → Diagram showing network security layers
